🤖 AI Recruitment Automation System

An AI-powered recruitment workflow built with n8n, OpenAI, Airtable, Gmail, and Google Calendar to automate candidate screening, resume scoring, interview scheduling, and recruiter notifications.

---

 📌 Business Problem

Recruitment teams spend significant time manually reviewing resumes, scoring candidates, scheduling interviews, and communicating with applicants.

These repetitive tasks reduce productivity and slow down the hiring process.

---

💡 Solution

This workflow automates the recruitment pipeline using AI.

The system:

- Receives candidate information
- Uses AI to evaluate resumes
- Scores candidates automatically
- Stores candidate data in Airtable
- Schedules interviews
- Sends Gmail notifications
- Keeps recruitment records organized

---

⚙️ Workflow Overview

```
Candidate
      │
      ▼
Application Submitted
      │
      ▼
AI Resume Scoring
      │
      ▼
Candidate Database (Airtable)
      │
      ▼
Interview Scheduling
      │
      ▼
Gmail Notification
```

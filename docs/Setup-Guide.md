# Setup Guide

## Requirements

- n8n
- OpenAI API Key
- Airtable Account
- Gmail Account
- Google Calendar

## Installation

1. Import the workflow into n8n.
2. Configure OpenAI credentials.
3. Connect Airtable.
4. Configure Gmail credentials.
5. Configure Google Calendar.
6. Test the workflow using sample candidate data.

The workflow is now ready for deployment.
